3Contract Note flow — date step, note-list card, per-note delivery, email-all, errors

Contract notes are per-trading-day documents — a range returns a list, not one file. New primitive: the 3b note-list card (paginated 10-at-a-time, month dividers, per-row download, segment badge only when a day has two notes). Calendar: 2018 floor, cap = today, no max range. All limits (page size 10, narrow-nudge threshold 50) are remote-config. No password on these PDFs. No invoice numbers surfaced. Delivery per note: PDF here or email; bulk = email all only — no download-all.
